# PveLauncher
A simple python eve launcher, supporting Two Factor Authentication and Evepath per account

### Features

 * TwoFactor Authentification
 * Mail Code support
 * ~~Update check~~
 * Serverstatus (using eve api)
 * Encrypted passwords (Masterpassword)
